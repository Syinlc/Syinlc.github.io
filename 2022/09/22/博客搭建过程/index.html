记录一下自己搭建博客的过程

前期准备

根据准备工具 - hexo-script使用文档 (gitbook.io)完成博客的搭建，其中需要注意的问题有两个

  1. 是github在创建仓库时默认生成main分支，需要将hexo根目录（_config.yml）配置修改为main

    deploy:
  type: git
  repo: git@github.com:Syinlc/Syinlc.github.io.git
  branch: main

  2. npm下载贼慢，建议换成国内淘宝镜像npm使用国内淘宝镜像的方法 - 水火和牛 - 博客园 (cnblogs.com)

    npm config set registry https://registry.npm.taobao.org

    2. 配置_config.yml

      buble: /libs/others/buble.js

  enable: true

    3. 在 themes>hexo-theme-matery>layout>layout.ejs 这的 body 标签中添加下面的代码

      <!-- 冒泡 -->
<% if (theme.buble.enable) { %>
    <script type="text/javascript">

  • 添加Live2D二次元老婆~(21条消息) Hexo添加Live2D看板娘最新教程_飞萤乱入帘的博客-CSDN博客_hexo live2d

    1. 有依赖需要先卸载，package.json是否有“hexo-helper-live2d”

      npm uninstall hexo-helper-live2d

    2. 安装依赖

      npm install --save hexo-helper-live2d

    3. 下载model文件

      npm install live2d-widget-model-shizuku

    4. 添加live2d到_config.yml配置文件中

      ## Live2D看板娘
live2d:
  enable: true
  pluginModelPath: assets/
  model:
    #模板目录，在node_modules里
    use: live2d-widget-model-shizuku  
  display:
    position: right
    width: 300 
    height: 600
  mobile:
    # 在手机端显示
    show: false   
  rect:
    opacity:0.7
